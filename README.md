# Todo App
Todo application built with React - add, edit, save, and remove tasks!

## Installation

Install Node Packages is required to run the application.

Navigate to project folder in the terminal and run the following:

```bash
$ npm install
```
Then run the server:

```bash
$ npm start
```

## Technologies Used
- React
- CSS Flexbox
- Font Awesome
- Netlify
